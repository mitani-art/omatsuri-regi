# 🏮 お祭りかんたんレジ＆カウンター

小学校のお祭り・バザー・地域イベント用の、スマホで使えるかんたんレジアプリです。
お年寄りや子どもでも迷わず使える大きなボタン設計で、**一度開けば完全オフラインで動きます**。

## 機能

- 🍧 **定額商品モード**：8つの金額ボタンをタップするだけ（金額は自由に変更可能）
- 🔢 **自由金額モード**：テンキーで任意の金額を入力（バザー・募金向け）
- 🙋 **無料カウンターモード**：人数だけをカウント（無料体験・展示向け）
- 💴 **おつり自動計算**：お預かりワンタッチでおつりを特大表示
- ↩ **取り消し機能**：押し間違えても1つ前に戻せる
- 📋 **集計コピー／📨 LINE送信**：閉店後の本部報告がワンタップ
- 💾 売上・設定は端末に自動保存（ブラウザを閉じても消えません）

## GitHub Pagesでの公開手順

1. このリポジトリの **Settings → Pages** を開く
2. **Branch** を `main`（フォルダは `/root`）にして **Save**
3. 数分待つと `https://ユーザー名.github.io/リポジトリ名/` で公開されます
4. このURLをLINEで各ブース担当者に送ってください

## 当日までの準備（ブース担当者向け）

1. 送られてきたURLをスマホで開く
2. ブラウザのメニューから **「ホーム画面に追加」** する
   - iPhone（Safari）：共有ボタン →「ホーム画面に追加」
   - Android（Chrome）：メニュー（︙）→「ホーム画面に追加」
3. ホーム画面の🏮アイコンから起動し、⚙で金額とブース名を設定
4. **電波のある場所で一度開いておけば、当日は圏外でも動きます**

## アプリを更新するとき（管理者向け）

`index.html` を修正したら、`sw.js` の先頭にある `CACHE_NAME` の数字を1つ上げてください
（例：`omatsuri-regi-v1` → `omatsuri-regi-v2`）。
各スマホが次にネットに繋がったとき、自動で新しい版に入れ替わります。

## ファイル構成

| ファイル | 役割 |
|---|---|
| `index.html` | アプリ本体（HTML・CSS・JSすべて入り） |
| `manifest.json` | ホーム画面追加時のアプリ名・アイコン設定 |
| `sw.js` | オフライン動作用のService Worker |
| `icon-192.png` / `icon-512.png` / `apple-touch-icon.png` | アプリアイコン |

## プライバシー

売上データは各スマホの中にのみ保存され、サーバーには一切送信されません。
